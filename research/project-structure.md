## Purpose of Each Folder

### ai-model
Contains AI model training, prediction, and saved models.

### backend
Handles API requests, image upload, and communication with the AI model.

### frontend
Contains website UI files including HTML, CSS, JavaScript, and images.

### dataset
Stores training, validation, and testing datasets.

### research
Contains research documents, planning, and analysis.

### docs
Project documentation and technical notes.

### assets
Icons, logos, diagrams, and design resources.

### screenshots
Stores screenshots for GitHub README and presentations.

### ppt
Hackathon presentation files.

### report
Final project report and documentation.
