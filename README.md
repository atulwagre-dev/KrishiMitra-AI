# 🌱 KrishiMitra AI

> AI-powered Offline Crop Disease Detection & Sustainable Farming Assistant

## 📌 Overview

KrishiMitra AI is an AI-powered web platform designed to help farmers detect crop diseases from smartphone images and receive sustainable farming recommendations, even in low-connectivity rural areas.

## 🎯 Problem Statement

Develop a computer vision tool for early crop disease and pest detection using smartphone images.

## 🌾 Theme

Agriculture & Allied

## ✨ Vision

Empowering every farmer with AI, anytime and anywhere.
